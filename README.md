# Repository Demo

Instalación, configuración y primeras actividades con Git.



# Guia basica

Buscamos si hay cambios en el remoto, en caso afirmativo nos los bajamos automaticamente:
 git pull

Editamos el archivo desde nuestro software preferido (Editor de texto, imagenes, etc.).

Añadimos el o los archivos que hayamos modificado para subirlos:
 git add "archivo.txt"

Añadidos un comentario:
 git commit -m "Estoy modificando archivo.txt"

Realizo la subida de los cambios realizado:
 git push


# Actividades

- Creando los repositorios remoto y local [OK]
- Trabajandocon el repositorio [OK]
- Conectando con Github [OK]
- Actualiza los datos [OK]
